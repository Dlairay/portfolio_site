AI controlled Turret

Project Description

This project was for a hackathon to design the best performing AI program to control a sentry turret. My role was to engineer the NLP module that converted human transcriptions into standardised instructions. I used a lightweight LLM model combined with pre and post regex processing and pydantic's schema, to built a blazingly fast model capable of doing function calling with near perfect accuracy, which directly contributed to my team's 1st runner up position.
